# Connect

Read [Open Knowledge Format v0.1](../references/SPEC.md) completely before interpreting or changing Knowledge Bundle artifacts.

1. **Inspect every side.** Read each concept participating in the proposed relationship and the passages that support it.
   - **Complete when:** every proposed endpoint and its relevant evidence have been inspected.
2. **Classify the relationship.** Apply the shared relationship rule in `SKILL.md` to every proposed connection.
   - **Complete when:** every proposed relationship has a support classification and a concise rationale.
3. **Represent supported connections.** Write the classified connections using the shared internal-link rule.
   - **Complete when:** every supported relationship is linked consistently, every remaining candidate is reserved for the report, and no known affected link is stale.
