# Revise

Read [Open Knowledge Format v0.1](../references/SPEC.md) completely before interpreting or changing Knowledge Bundle artifacts.

1. **Locate the change surface.** Read every target concept, its index context, and connected concepts that the requested change may affect.
   - **Complete when:** the requested change and every clear downstream effect have an identified file or an explicit reason no file is affected.
2. **Apply the authorized revision.** Make the requested semantic or structural change using the supplied evidence. Preserve unresolved competing claims and collect them for the report.
   - **Complete when:** every authorized change is applied, and every ambiguity remains unchanged and is reserved for the report.
3. **Restore structural consistency.** Update affected timestamps, index entries, descriptions, and internal links using the shared writing rules in `SKILL.md`.
   - **Complete when:** every changed concept agrees with its navigation and supported connections, with no known stale structural effect left behind.
